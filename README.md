InstaLite is a full-stack Instagram-like social media application built with MERN stack. It allows users to create accounts, upload photos, like & comment on posts, follow/unfollow users, receive real-time notifications, and manage profiles with secure authentication. The app includes responsive UI, cloud image storage, and JWT-based authorization.

**Key Features**

User authentication (Signup/Login using JWT + bcrypt)
Upload & delete posts with image storage
Like / Unlike, Comments system
Follow / Unfollow users
Real-time notifications & activity system
User profile page with post lists
Explore feed & personalized timeline
Responsive UI similar to Instagram
Admin panel to manage users
Dark/Light theme support

**Tech Stack**

Frontend: React.js, Redux Toolkit, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Image Storage: Cloudinary / Firebase
Authentication: JWT, Bcrypt
